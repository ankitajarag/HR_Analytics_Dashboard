# HR_Analytics_dashboard
The HR Analytics Dashboard is a Power BI project designed to analyze and visualize key workforce metrics within an organization. This interactive dashboard helps HR departments monitor employee performance, attrition, job satisfaction, demographics, and other critical KPIs — enabling data-driven decisions to improve retention and productivity.

 Objectives

Identify trends in employee attrition and satisfaction.

Understand workforce distribution by department, gender, and job role.

Monitor employee performance ratings and experience levels.

Provide actionable insights for HR strategy and policy decisions.

 Tools & Technologies

Power BI Desktop – For data modeling, analysis, and visualization

Excel / CSV Dataset – Source data (Employee-related metrics)

DAX & Power Query – Used for calculations, KPIs, and transformations

 Key Insights

Attrition rate across departments and job roles

Relationship between job satisfaction and performance

Employee distribution by age, gender, and education

Salary trends and experience correlation

Dashboard Features

Interactive slicers and filters

Drill-down visualizations for deeper insights

Dynamic KPIs and performance indicators

Data-driven story for HR management decisions
